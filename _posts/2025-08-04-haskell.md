シーザー暗号の章。indexOf関数の実装は、再帰を使用して該当する要素のインデックスを取得するものであり、非常にわかりやすい
indexOf :: Char -> [Char] -> Int
indexOf ch (x : xs) = if x == ch then 0 else 1 + indexOf ch xs
